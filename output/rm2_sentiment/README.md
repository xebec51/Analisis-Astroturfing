# RM2 Sentiment Output

This folder separates compact final reporting outputs from human validation files and frozen model artifacts.

| path | status | notes |
|---|---|---|
| `final/` | CANONICAL_FINAL_OUTPUT | Compact final sentiment outputs, reports, Gephi attributes, presentation files, and refreshed final PNG visualizations. |
| `final/tables/` | CANONICAL_FINAL_OUTPUT | Final account, HCC, brand, distribution, and HCC-vs-Non-HCC summaries. |
| `final/visualisasi/` | CANONICAL_FINAL_OUTPUT | Final PNG visualizations regenerated by `notebooks/rm2/02_rm2_sentiment_analysis.ipynb`. |
| `final/gephi/` | CANONICAL_FINAL_OUTPUT | Final Gephi-ready sentiment attributes. |
| `model/frozen/` | FROZEN_RESEARCH_ARTIFACT | Frozen model, threshold, evaluation lock, and compact locked-test metrics. |
| `validation/human_v1/` | DEVELOPMENT_PROVENANCE | Original human validation package retained for provenance. |
| `validation/human_v2/` | DEVELOPMENT_PROVENANCE | Final human annotation, replacement, locked-test, and provenance files. |
| `legacy/v1/` | LEGACY_VISUAL_REFERENCE | Old PNG visualizations retained for visual comparison only; bulky V1 CSV tables are not tracked. |
| `legacy/exploratory/visualisasi/` | EXPLORATORY_OUTPUT | Wordcloud exploratory PNGs, not validation evidence. |

Important status:

- Final model status: `FINAL_MODEL_VALIDATED`.
- Final sentiment status: `FINAL_MODEL_VALIDATED_SENTIMENT_V2`.
- Threshold: `0.42`.
- Locked-test evaluation: `FINAL_LOCKED_TEST_EVALUATED_ONCE`.

Sentiment is an indicator of observed message orientation, not evidence of intent, payment, affiliation, control,
or causal influence.
