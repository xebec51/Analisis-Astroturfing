# RM2 Actor Type Output

This folder contains RM2 actor type outputs.

| path | status | notes |
|---|---|---|
| `tables/` | frozen canonical | Actor universe, actor type summaries, edge summaries, and validation tables. |
| `gephi/` | frozen canonical | Actor type and Community-Mass Gephi CSV exports. |
| `visualisasi/` | frozen canonical | PNG visualizations. |
| `audit/` | frozen canonical | Cleaning and validation audit tables. |
| `account_interaction/` | frozen canonical | Three-evidence Community-Mass account-level network. |
| `direct_interaction/` | legacy diagnostic | Reply-only diagnostic retained as provenance. |

Mass Actor is the residual actor universe after Individual Actor and Community Actor are assigned. LCN position
is stored separately from actor type.
