# RM1 Temporal Output

This folder contains the final RM1 temporal activity profile outputs. The main timezone is
`Asia/Jakarta` / WIB.

| path | status | notes |
|---|---|---|
| `tables/` | frozen canonical | Temporal summaries for all dataset, LCN, and HCC scopes. |
| `visualisasi/` | frozen canonical | PNG line charts by weekday, hour, and weekday-hour. |

Temporal outputs are descriptive activity profiles. They do not prove work schedules, commercial coordination,
or causal influence.
